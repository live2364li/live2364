Clash for Android 使用教程

0. 简介

Clash是一款用 Go语言 开发的支持 Linux/MacOS/Windows/Android 等多平台的代理工具，支持 ss/v2ray（不支持 ssr），支持规则分流（类似于 Surge 的配置）。

Clash(R)则是基于Clash开发，从而支持SSR节点，请注意SS和SSR是有区别的！

由于客户端长期更新中可能出现界面或功能与教程不符，我们建议您把本教程仅作为参考，具体需要根据自己使用过程中熟悉了解。

1. 下载客户端 下载 Clash 客户端

下载客户端并进行安装

启动客户端并进行下面操作

2. 设置订阅地址  Clash 订阅配置     ClashR 订阅配置  

点击上方按钮一键导入节点配置到ClashR

文件名填入 好家伙 - Cloud

然后点击右上角的 √ 完成导入

再次打开软件，进入 配置文件 选中刚刚导入的节点配置

无法一键导入? 试试手动设置订阅

先复制 Clash/ClashR 订阅地址

打开APP 进入 配置 - 选择 新配置 - URL

粘贴订阅地址到 URL 一栏, 名称填写 好家伙 - Cloud, 点击右上角保存

回到 配置文件 选中刚刚导入的节点配置

3. 开启代理并选择节点

点击 已停止 按钮将状态切换至运行中

点击 控制面板 按钮进入节点配置界面

根据规则设置节点即可
